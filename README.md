# Laravel E-commerce API

## 📌 Overview
This is a backend API for an E-commerce system built with Laravel.  
It includes authentication, product management, cart system, and order processing.

---

## 🚀 Features

### 🔐 Authentication
- Login
- OTP verification
- Password reset

### 👥 Roles & Permissions
- Admin / User roles using Spatie

### 🛍️ Products
- CRUD operations
- Slug generation
- Image upload
- Filtering & Pagination

### 🛒 Cart
- Add to cart
- Update quantity
- Remove items
- Calculate total

### 📦 Orders
- Checkout system
- Order creation with transaction
- Stock management
- Order items

### 👨‍💼 Admin
- View all orders
- Update order status

---

## ⚙️ Tech Stack
- Laravel
- MySQL
- Sanctum
- Spatie Permissions
